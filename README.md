  Digi Fin Literacy Africa Website

  Overview

The Digi Fin Literacy Africa Website is a responsive multi page web application developed as part of a Web Application Development project. The website represents Digi Fin Literacy Africa (DLA), an initiative committed to empowering African youth, women, students, entrepreneurs, and underserved communities through digital literacy and financial literacy education.

The platform serves as an informational and engagement hub where visitors can learn about the organization's mission, programs, events, educational resources, success stories, and opportunities for participation.

   

  Project Objectives

The primary objectives of this project are:

* Promote digital literacy across African communities.
* Improve financial literacy and responsible money management skills.
* Create awareness of digital entrepreneurship opportunities.
* Provide access to educational resources and learning materials.
* Showcase the impact of Digi Fin Literacy Africa initiatives.
* Demonstrate modern front end web development practices.

   

  Website Features

   Responsive Design

* Mobile first design approach.
* Fully responsive across desktop, tablet, and mobile devices.
* Bootstrap powered layouts and components.

   Multi Page Navigation

The website consists of the following pages:

1. Home
2. About Us
3. Programs
4. Resources
5. Events
6. Success Stories
7. Blog
8. Contact Us

All pages are interconnected through a responsive navigation menu and footer links.

   Interactive Features

* Contact form validation using JavaScript.
* Dark/Light mode toggle.
* Dynamic resource search/filter functionality.
* Bootstrap carousel and interactive components.

   Accessibility & Usability

* Semantic HTML5 structure.
* Responsive navigation bar.
* Accessible forms and image descriptions.
* User friendly interface and consistent branding.
